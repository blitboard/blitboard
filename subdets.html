Binary Trees



Structure

Structure of a Binary Tree in Object Oriented format

Traversing

Jumping from node to node and analyzing its value as per the requirements is called traversing a tree.

Size of a Tree

Size refers to the number of nodes present in a Binary Tree. Increment a counter at every traversal.

Height of a Tree

Height is the no. of levels of a tree.

This can be found out adding the height of larger subtree(again in a recursive function) to 1.

Diameter of a Tree

Diameter of a binary tree is defined as the stretched node to node distance, maximum in a tree. In simple terms, it is the left height, the root node and the right height.

Leaves of a Tree

Look out for the bottom most nodes of a tree.

Preorder Traversal

Jumping from node to node in left-node-right fashion.

Inorder Traversal

Jumping from node to node in a left-node-right fashion.

Postorder Traversal

Jumping from node to node in left-right-node pattern.

Preorder Traversal-Iterative

Traverse a binary tree in a node-left-right fashion without recursion(STACK required)

Inorder Traversal-Iterative

Traverse a binary tree in a left-node-right fashion without recursion(STACK required)

Postorder Traversal-Iterative

Traverse a binary tree in a left-right-node fashion without recursion(2 STACKS required.)

Preorder Traversal-Morris Traversal

Traverse a binary tree in a left-node-right fashion without recursion and without stack.

Inorder Traversal-Morris Traversal

Traverse a binary tree in a left-node-right fashion without recursion and without stack.

Postorder Traversal-Morris Traversal

Traverse a binary tree in a left-right-node fashion without recursion and without stacks.

Level Order Traversal

Level Order Traversal is a particular traversal order where you can start from the topmost node and proceed to the bottom nodes in left to right direction.

Reverse Level Order Traversal

Reverse Level Order Traversal is a particular traversal order where you can start from the bottom node and proceed to the top nodes in left to right direction.

Spiral Level Order Traversal

Spiral in Level Order Traversal in a binary tree referse to printing one level in left to right format and other in right to left, taking place alternatively

Level of a Node

Level of a node in a Binary Tree can be found out by an incremented value which is 1 initially at head always and adds 1 every time when moved

Stepped Inwards Traversal

Size refers to the number of nodes present in a Binary Tree. Increment a counter at every traversal.

Density of a Tree

Density of a Binary Tree is defined as the ratio of Size to the Height of a Tree. Size is where the number of nodes and Height is the no. of levels of a tree.

Incomplete Node

Nodes that remove the completeness of a tree, i.e. nodes that just have a left node or a right node but not both.

Nodes with no Sibling

Print all nodes that do not have another node connected to their parent node at same level in a Binary Tree
